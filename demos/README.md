# Demos (Invoice Processor)

## 60–90 second demo script (talk track)

1) **Problem**: "Invoices come in as PDFs. Someone has to manually type vendor, totals, line items, payment terms into accounting software. It’s slow and error-prone."  
2) **Show**: Drag & drop a PDF invoice into the app.
3) **Magic**: AI extracts vendor, dates, line items, totals, terms.
4) **Human-in-the-loop**: Make one edit inline (prove it’s safe + controllable).
5) **Export**: Download JSON/CSV.
6) **Outcome**: "This turns PDFs into clean structured data in seconds — and can feed QuickBooks/Xero/Sheets/CRMs next."
